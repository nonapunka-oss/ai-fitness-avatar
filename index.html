About This Project

This project presents an end-to-end AI-powered avatar creation workflow, demonstrating how multiple artificial intelligence tools can be integrated into a cohesive digital production pipeline.

The visual avatar was generated using a large language model (LLM). The voice recording was produced using AI-assisted speech synthesis technology. The animated talking avatar video was created using HeyGen by synchronizing the generated voice with the avatar image.

The final interactive website was developed using HTML, CSS, and JavaScript, and deployed via GitHub Pages using an automated GitHub Actions workflow.

This project highlights the practical integration of generative AI tools, media synthesis technologies, and front-end development to create a fully functional AI-driven presentation experience.

Workflow Overview

Avatar image generation using LLM-based AI tools
Voice creation using AI speech synthesis
Video production with synchronized avatar animation
Frontend development (HTML, CSS, JavaScript)
Deployment via GitHub Pages with CI/CD automation
